# Modernised PSP Design System

Design system for the Amazon Pay Payment Selection Page (PSP).

**Live**: [sbansalh.github.io/PSP-Design-System](https://sbansalh.github.io/PSP-Design-System/)

## Files

| File | Purpose |
|------|---------|
| `index.html` | Interactive design system docs (main deliverable) |
| `design-tokens.json` | Structured tokens for code / Tokens Studio |
| `psp-modernised.html` | Pixel-accurate HTML mockup of PSP screen |
| `figma-plugin-script.js` | Figma plugin to create components + styles |
| `design.md` | Architecture decisions and rationale |
| `skill.md` | Kiro skill definition for PSP work |

## Figma Source

[Modernised PSP toolkit](https://www.figma.com/design/XoqbHriFr2Efq18TBPG6VQ/Modernised-PSP-toolkit?node-id=9-1857)

## Key Specs

- Font: Amazon Ember (400, 700)
- Grid: 4px base, 360px frame
- Buttons: From Tuxedo Mobile or RIO (not in this system)
- Preselection: Best Offer > Previously Used > Single > None
- Section order: Fixed (Recommended > UPI > Cards > More Ways > Gift Cards)
